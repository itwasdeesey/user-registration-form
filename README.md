# Звіт про виконану роботу

## 1. Тема та мета роботи

**Тема:** Розробка форми реєстрації користувача з валідацією даних

**Мета роботи:** Створити інтерактивну веб-форму реєстрації користувача з реалізацією клієнтської валідації всіх обов'язкових полів та відображенням повідомлень про помилки введення.

**Завдання:**
- Розробити форму реєстрації з полями: логін, ім'я, електронна пошта, пароль, номер телефону, дата народження
- Реалізувати валідацію кожного поля згідно з вимогами:
  - Логін: мінімум 5 символів
  - Ім'я: тільки літери
  - Пароль: мінімум 8 символів, хоча б одна велика літера та одна цифра
  - Електронна пошта: наявність символу @ та доменного імені
  - Дата народження: перевірка формату та відсутність майбутньої дати
- Забезпечити відображення помилок валідації під кожним полем
- Створити сучасний адаптивний дизайн форми

## 2. Приклад коду

### Приклад функції валідації

```javascript
function validateLogin() {
    const login = loginInput.value.trim();
    const errorElement = document.getElementById('loginError');
    
    if (login.length < 5) {
        showError(loginInput, errorElement, 'Логін повинен містити мінімум 5 символів');
        return false;
    }
    
    clearError(loginInput, errorElement);
    return true;
}

function validatePassword() {
    const password = passwordInput.value;
    const errorElement = document.getElementById('passwordError');
    
    if (password.length < 8) {
        showError(passwordInput, errorElement, 'Пароль повинен містити мінімум 8 символів');
        return false;
    }
    
    if (!/[A-ZА-ЯЁ]/.test(password)) {
        showError(passwordInput, errorElement, 'Пароль повинен містити хоча б одну велику літеру');
        return false;
    }
    
    if (!/[0-9]/.test(password)) {
        showError(passwordInput, errorElement, 'Пароль повинен містити хоча б одну цифру');
        return false;
    }
    
    clearError(passwordInput, errorElement);
    return true;
}
```

## 3. Висновки

В результаті виконання роботи було створено повнофункціональну форму реєстрації користувача з наступними досягненнями:

1. **Реалізовано всі вимоги до валідації:**
   - Валідація логіну (мінімум 5 символів)
   - Валідація імені (тільки літери)
   - Валідація пароля (мінімум 8 символів, велика літера та цифра)
   - Валідація електронної пошти (@ та доменне ім'я)
   - Валідація дати народження (формат та перевірка на майбутнє)

2. **Створено зручний інтерфейс:**
   - Горизонтальний layout з двома колонками для компактності
   - Валідація в реальному часі при втраті фокусу поля
   - Візуальна індикація помилок (червона рамка) та валідних полів (зелена рамка)
   - Анімація "shake" для полів з помилками

3. **Забезпечено адаптивність:**
   - Форма адаптується під різні розміри екранів
   - На мобільних пристроях поля автоматично переходять в один стовпець

4. **Сучасний дизайн:**
   - Використано технологію glassmorphism (скляний ефект)
   - Анімований фон з декоративними елементами
   - Плавні переходи та інтерактивні ефекти

Проект демонструє навички роботи з HTML, CSS та JavaScript, вміння реалізувати клієнтську валідацію форм та створювати сучасний адаптивний веб-інтерфейс.
